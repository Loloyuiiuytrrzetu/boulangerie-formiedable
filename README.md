# Maison de Rozières

Site vitrine de la **Maison de Rozières**, boulangerie & pâtisserie artisanale
située à Jardivillage Jabrun, Baie-Mahault, Guadeloupe.

## Aperçu

Site statique (HTML / CSS / JS) — thème **rouge cerise**.

- `index.html` — page unique avec sections Accueil, À propos, Spécialités, Horaires, Avis, Contact
- `styles.css` — thème cerise, responsive
- `script.js` — menu mobile, statut d'ouverture en direct, horloge, animations
- `assets/` — logo et photos

## Lancer en local

Ouvrir simplement `index.html` dans un navigateur, ou servir le dossier :

```bash
python3 -m http.server 8000
```

Puis ouvrir http://localhost:8000.
